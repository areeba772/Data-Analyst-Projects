Data Analysis Portfolio
Welcome to my data analysis portfolio! This repository showcases my projects, ranging from Excel-based dashboarding to SQL-driven business intelligence.

🚀 Project 01: E-Commerce Sales Analysis
Project Overview
This project involves cleaning and analyzing an e-commerce sales dataset to uncover key performance trends. The objective was to transform raw sales data into actionable insights through visualization and interactive dashboarding using Microsoft Excel.

Key Features
Data Cleaning: Processed raw transactional data to ensure consistency and accuracy.

Data Modeling: Organized data to enable efficient filtering and aggregation.

Interactive Dashboard: Created an intuitive dashboard featuring slicers for dynamic exploration by Region and Category.

Visualizations
Category vs. Sales: Displays total sales distribution across different product categories.

Month vs. Sales: Tracks sales trends over time to identify seasonal patterns.

Region vs. Profit: Highlights the most profitable geographical regions.

Category vs. Month vs. Sales: A multi-dimensional view breaking down sales performance.

Recommendations
Optimize Inventory for High-Performing Regions: Prioritize stock allocation and marketing budgets for regions showing higher profitability to maximize return on investment.

Targeted Seasonal Promotions: Utilize sales trends to identify peak months and plan proactive campaigns to capture maximum revenue.

Tools Used
Microsoft Excel: Pivot tables, data cleaning, and visualization.

☕ Project 02: Monday Coffee Sales Analysis
Project Overview
This project focuses on analyzing coffee sales data to derive business insights for Monday Coffee. By examining customer behavior, city-wise sales performance, and market potential, this analysis helps in making data-driven decisions regarding expansion and marketing strategies.

Key Features
Database Design: Created a normalized relational database schema (Customers, Products, City, Sales).

Data Cleaning: Performed schema adjustments and ensured data integrity using foreign key constraints.

Business Intelligence: Developed complex SQL queries (using DENSE_RANK, LAG, and WITH clauses) to solve real-world business problems.

Key Business Insights
Market Penetration: Calculated potential coffee consumers in each city (assuming 25% of population).

Revenue Analysis: Tracked 2023 quarterly revenue trends.

Operational Efficiency: Analyzed "Average Rent per Customer" to identify cost-effective cities.

Recommendations
Pune: Highly profitable market due to highest total revenue and low average rent per customer.

Delhi: Greatest market potential with 7.7 million estimated coffee consumers.

Jaipur: Excellent balance of high customer volume and very low rent costs (156 per customer).

Technologies Used
SQL (MySQL/PostgreSQL): Database modeling, data manipulation, and complex analytical querying.
